Using the Material Theme
----
使用材料主题
----

The new material theme provides:

- System widgets that let you set their color palette

- Touch feedback animations for the system widgets

- Activity transition animations

新材料主题将提供：

- 可设置调色板的系统组件

- 系统组件的触摸反馈动画

- Activity转换动画
